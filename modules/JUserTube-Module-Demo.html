This page shows all the layouts of JUserTube Module. At this moment 4 layouts are available. You can Adjust the following settings from module parameter.

Sync Interval Sorting (Playlist Only) Number of thumb to show Number of videos to sync Thumbs in a row on large display (grid layout) Thumbs in a row on small display (grid layout) Thumbs padding (grid layout) Show/Hide Title (grid layout) Show/Hide Overlay Icon (grid layout + description layout) Thumbnail corner rounding (grid layout + description layout) Thumbnail Shadow (grid layout + description layout) Truncate title after certain character (grid layout + description layout) Truncate description after certain character (description layout) Thumb:Description width ratio (description layout) Height, sliding speed, initial video (slider and slider+video layout)

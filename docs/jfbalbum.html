Installation/Setup Related

Installation (Joomla 2.5.x and 3.x)
  1. Login to admin area.
  2. Go to -> Extensions -> Extension Manager.
  3. Under 'Upload Package File' Point the downloaded package and click Upload & Install.
  4. After installation setup a Module or Menu Item ( Album or Gallery ).
  5. If you're upgrading from an older version to version 4.1, you'll also have to remove 'JFBAlbum resource loader' plugin (Also uninstall the JFBAlbum component and JFBAlbum module for a cleaner install)
  6. On some server it may not show anything without app id and secret. You can simply create an app from Facebook Developers to get app id and secret. After getting them put it on appropriate fields for each album/gallery parameters (module+menu item)
  7. If you see any javascript error after installing this, That's probably a jQuery problem which can easily be solved by jQuery Easy plugin or 'SrizonModifier' plugin (provided with this package). Also you can Send us an email with admin access. We'll fix it for you.
Module Setup: Album Only (Joomla 2.5.x & Joomla 3.x)
  1. Go to Admin area -> Extensions -> Module Manager.
  2. Search/Filter for JFBAlbum to find the module (This won't work if you change the module title).
  3. OR from 'Select Type' dropdown choose the type 'JFBAlbum' to list all the instance of JFBAlbum.
  4. OR click New and under 'Select a Module Type:' select JFBAlbum to add a new instance of the module.
  5. Set the Details and Menu Assignment like any other module.
  6. Set the parameters of the module.
  7. Set The AlbumIDs (How to get?) and other parameters.
Album (Menu Item) Setup (Joomla 2.5.x & Joomla 3.x)
  1. Go to Admin area -> Menus -> Select the menu where you want to add the new menu item.
  2. Click 'New'.
  3. Click Select beside Menu Item Type.
  4. From The 'Select a Menu Item Type:' popup Locate and click 'JFBAlbum - Fullpage'.
  5. Set the Details like any other menu.
  6. Set the parameters of the menu (Joomla 2.5.x).
  7. Set JFBAlbum related parameters from 'Advanced Options' tab (Joomla 3.x).
  8. Set The Album IDs (How to get?) and other parameters.
Gallery (Menu Item) Setup (Joomla 2.5.x & Joomla 3.x)

Gallery view is a 2 level view for albums (just like facebook photos page). It works only with public fan.

  1. Go to Admin area -> Menus -> Select the menu where you want to add the new menu item.
  2. Click 'New'.
  3. Click Select beside Menu Item Type.
  4. From The 'Select a Menu Item Type:' popup Locate and click 'JFBAlbum - Gallery'.
  5. Set the Details like any other menu.
  6. Set the parameters of the menu (Joomla 2.5.x).
  7. Set JFBAlbum related parameters from 'Advanced Options' tab (Joomla 3.x).
  8. Set the PageID (How to get?) and other parameters.
Getting Album ID
  1. Visit your Facebook Page.
  2. Go to Photos->Albums.
  3. Go to an album.
  4. The URL (link) of the album should be like this 
    https://www.facebook.com/media/set/?set=a.434654226025.235247.64822581025&type=3
  5. The number after a. is the album id. For the link above it's 
    434654226025
  6. You may also enter the whole URL (link) as album it. However, it may not work if the URL structure changes.
Getting Page ID (for creating a Gallery)
  1. If you page URL (link) is 
    http://www.facebook.com/pages/sometext/number
    then the PageID is the 
    number
    You'll get this type of URL (link) if you haven't chosen an unique username for your page
  2. if your page URL is 
    http://www.facebook.com/name
    then the PageID is the 
    name

Troubleshooting Related

Installation Failed!

Make Sure the directory permissions are ok.
Setup Ok. But nothing shows on the frontend or the layout/lightbox is broken

Hard Way: Make Sure all of the following are checked:

  1. You already tested your target server/site with the server test module.
  2. The album id or page id is from a public page on Facebook (not a personal profile where you add friend or a group).
  3. Your page is published.
  4. Your page doesn't have any country/age restriction. (See how to fix)
  5. Your page isn't categorized under alcohol/nightclub etc adult category.
  6. There's no javascript error on your page (most likely to happen on J 2.5.x). If there is, 99% time it can be solved using jQuery Easy plugin
  7. There's also a plugin called 'srizonmodifier' included with this package which is able to add/remove lines of codes and help solving duplicated jQuery issue. Use that if necessary

Easy Way: Send us an email with admin access, menu link to the album page and facebook page/album link
I just updated something on Facebook album but it's not updating on my site

You need to wait for the next sync. Sync interval can be adjusted from the backend parameter. You can also do a force re-sync to see immediate result by deleting Joomla cache.
How to remove restrictions from a Facebook Page
  1. Visit the facebook page as an admin user.
  2. Click edit page -> edit settings.
  3. Remove any restrictions (age / country etc).
  4. If the page is categorized under bar / alcohol related then remove as well.
  5. After saving the changes try to load your page as 'logged-out' user (without loggin in) if it loads then it's ready.
  6. Force re-sync (admin menu->components->jfbalbum->sync now).

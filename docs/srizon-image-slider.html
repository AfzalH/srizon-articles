Installation/Setup Related

Installation (Joomla 2.5.x and 3.x)
  1. Login to admin area
  2. Go to -> Extensions -> Extension Manager
  3. Under 'Upload Package File' Point the downloaded package and click Upload & Install
  4. After installation setup a Module
Module Setup: Album Only (Joomla 2.5.x & Joomla 3.x)
  1. Go to Admin area -> Extensions -> Module Manager.
  2. Search/Filter for Srizon Image Slider to find the module or from 'Select Type' dropdown choose the type 'Srizon Image Slider' to list all the instance of Srizon Image Slider
  3. OR click New and under 'Select a Module Type:' select Srizon Image Slider to add a new instance of the module
  4. Set the Details and Menu Assignment like any other module.
  5. Set the parameters of the module.

Troubleshooting Related

Installation Failed

Make Sure the directory permissions are ok
Setup Ok. But slider is not working

Make Sure there's no JavaScripit error. It's required that you have a JavaScript error free frontend or the module may not work

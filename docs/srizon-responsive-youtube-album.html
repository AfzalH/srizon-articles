Installation/Setup Related

Installation (Free Version)

Method 1:

  1. Login to your WordPress Admin
  2. Go to Plugins Menu and click Add New
  3. Search with the word 'srizon' and you'll see all our plugins
  4. Install and Activate

Method 2:

  1. Download the zip and up-zip it
  2. Upload srizon-responsive-youtube-album folder to the /wp-content/plugins/ directory
  3. Activate the plugin through the 'Plugins' menu in WordPress

Method 3:

  1. Download the zip
  2. Go to Plugins menu from your WordPress admin
  3. Click Add New and select upload option
  4. Upload, Install and Activate
Installation (Pro Version)

Deactivate and delete the free version and install using Method 2 or 3 above
Setting Up an Album
  1. Go to your WordPress admin and click on the menu 'YouTube Album'
  2. Click on Add New and fill out different parameters
  3. Most important parameter is YouTube channle or playlist ID. (click here to see how to get it)
  4. Save and Go back to album listing to get the shortcode
  5. Copy the shortcode and paste into a page/post
Video Source Setup
  1. If the URL (link) of your channel is 
    http://www.youtube.com/trailers
    or 
    http://www.youtube.com/user/trailers
    then your user/channel id is 
    trailers
  2. If the URL (link) for the playlist (Pro Only) is 
    https://www.youtube.com/playlist?list=PLjCq3Gx18tZvfcwjy7ZwGjrX7X5m41vpv
    then the playlist id is 
    PLjCq3Gx18tZvfcwjy7ZwGjrX7X5m41vpv

Troubleshooting Related

Setup Ok but the shortcode is not showing anything

Your server may not be compatible for using YouTube API. Ask your hosting admin to enable remote connectivity features of PHP (for example allow_url_fopen or cURL library)
Playlist is not sorting according to my selection (pro version)

You need to wait for the next re-sync or do a manual re-sync
Lightbox and/or Slider (pro version) is not working

Most probably a JavaScript error occurred due to another extension or your template. Make sure your frontend is JavaScript error free. In a plug and play system it's important to check for JavaScript error after installation of each extension. You'll have to disable the conflicting extension if you're not able to resolve the conflict.

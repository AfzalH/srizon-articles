Installation/Setup Related

Installation (Free Version)

Method 1:

  1. Login to your WordPress Admin.
  2. Go to Plugins Menu and click Add New.
  3. Search with the word 'srizon' and you'll see all our plugins.
  4. Install and Activate.

Method 2:

  1. Download the zip and up-zip it.
  2. Upload srizon-facebook-album folder to the /wp-content/plugins/ directory.
  3. Activate the plugin through the 'Plugins' menu in WordPress.

Method 3:

  1. Download the zip
  2. Go to Plugins menu from your WordPress admin.
  3. Click Add New and select upload option.
  4. Upload, Install and Activate.
Installation (Pro Version) - Upgrading from free version

Deactivate and delete the free version and install using Method 2 or 3 above.
Installation (Pro Version) - Upgrading from older pro version
  • Delete this directory 
    site-root/wp-content/plugins/srizon-facebook-album-pro
  • Also delete this directory if exists 
    site-root/wp-content/plugins/srizon-facebook-album
  • install using Method 2 or 3 above.
Setting Up an Album
  1. Go to your WordPress admin and click on the menu 'FB Album' and then click on 'Albums'.
  2. Click on Add New.
  3. Set The Album IDs (How to get?) and other parameters.
  4. Save and Go back to album listing to get the shortcode.
  5. Copy the shortcode and paste into a page/post.
Setting Up a Gallery
  1. Go to your WordPress admin and click on the menu 'FB Album' and then click on 'Galleries'.
  2. Click on Add New.
  3. Set the PageID (How to get?) and other parameters.
  4. Save and Go back to album listing to get the shortcode.
  5. Copy the shortcode and paste into a page/post.
Getting Album ID
  1. Visit your Facebook Page.
  2. Go to Photos->Albums.
  3. Go to an album.
  4. The URL (link) of the album should be like this 
    https://www.facebook.com/media/set/?set=a.434654226025.235247.64822581025&type=3
  5. The number after a. is the album id. For the link above it's 
    434654226025
  6. You may also enter the whole URL (link) as album it. However, it may not work if the URL structure changes.
Getting Page ID (for creating a Gallery)
  1. If you page URL (link) is 
    http://www.facebook.com/pages/sometext/number
    then the PageID is the 
    number
    You'll get this type of URL (link) if you haven't chosen an unique username for your page
  2. if your page URL is 
    http://www.facebook.com/name
    then the PageID is the 
    name

Troubleshooting Related

Setup Ok but the shortcode is not showing anything

Your server may not be compatible for using Facebook API. Ask your hosting admin to enable remote connectivity features of PHP (for example allow_url_fopen or cURL library).
Lightbox is not working

Most probably a JavaScript error occurred due to another extension or your template. Make sure your frontend is JavaScript error free. In a plug and play system it's important to check for JavaScript error after installation of each extension. You'll have to disable the conflicting extension if you're not able to resolve the conflict.

Pagination and 'Back to Gallery' links are not working

Those links are designed to take you directly to the album/gallery area with a #albumid appended to the link. Some smooth scroll scripts seems to block the normal behaviour of such link and just scrolls it to the #albumid area. So you can do one of the following

  1. Change the smooth scroll script
  2. Change some code in 'Srizon Facebook Album' plugin as described below

Change needed for 'Back to gallery' link: Open the file tmpl/gallery.php and change the following line of code (line 7)

$blink = $url.'#'.$scroller_id;

to

$blink = $url;

Change needed for Pagination links: Open the file lib/srizon_functions.php and change the following lines of code (line 48-52)

if (strpos($url, '?')) {
    return $paginator->page_links($url . '&', '#' . $scroller_id);
} else {
    return $paginator->page_links($url . '?', '#' . $scroller_id);
}

to

if (strpos($url, '?')) {
    return $paginator->page_links($url . '&');
} else {
    return $paginator->page_links($url . '?');
}

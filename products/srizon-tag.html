About this product:

This Joomla Module displays customized Tag-Cloud on your Joomla site. You can setup up-to 30 tags and their corresponding links (it will be linked to search results if link is not defined for a tag). You can also specify the size and color for each tag. You can select any web safe font from a drop-down (common for all tags in one module instance). There are more configuration options (speed, direction, opacity, outline, shape etc.).

This extension is based on TagCanvas script.

System Requirements

  • Joomla installation (at least version 2.5.11) with JavaScript error free frontend
  • Modern web browser (latest version of any browser) that can render HTML5 canvas elements
